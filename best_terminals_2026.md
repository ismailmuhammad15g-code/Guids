## Setup and Configuration Guide

### Step 1: Choose Your Shell

```bash
# Check current shell
echo $SHELL

# Install zsh (recommended)
# macOS: brew install zsh
# Ubuntu: sudo apt install zsh

# Change default shell
chsh -s $(which zsh)
```

### Step 2: Install Oh My Zsh

```bash
# Install Oh My Zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# Popular plugins
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
```

### Step 3: Install Nerd Fonts

```bash
# macOS
brew install --cask font-jetbrains-mono-nerd-font

# Linux
wget https://github.com/ryanoasis/nerd-fonts/releases/latest/download/JetBrainsMono.zip
unzip JetBrainsMono.zip -d ~/.fonts/
fc-cache -fv
```

## Advanced Tips and Tricks

### 1. Master Keyboard Shortcuts

```bash
# Universal
Ctrl+C          # Cancel current command
Ctrl+L          # Clear screen
Ctrl+R          # Reverse search history
Ctrl+D          # Exit terminal
Ctrl+Z          # Suspend process
Ctrl+A          # Go to beginning of line
Ctrl+E          # Go to end of line
```

### 2. Create Useful Aliases

Add to your `~/.zshrc`:
```bash
# Navigation
alias ..='cd ..'
alias ...='cd ../..'

# Git shortcuts
alias gs='git status'
alias ga='git add'
alias gc='git commit'
alias gp='git push'

# System
alias ll='ls -la'
alias du='du -h'
alias df='df -h'
```

### 3. Use Tmux for Session Management

```bash
# Install tmux
# macOS: brew install tmux
# Linux: sudo apt install tmux

# Basic commands
tmux                    # Start new session
tmux new -s name        # Create named session
tmux ls                 # List sessions
tmux attach -t name     # Attach to session
```

### 4. Backup Your Configuration

```bash
#!/bin/bash
BACKUP_DIR="$HOME/terminal-backup-$(date +%Y%m%d)"
mkdir -p $BACKUP_DIR

cp ~/.zshrc $BACKUP_DIR/
cp -r ~/.oh-my-zsh $BACKUP_DIR/
cp ~/.config/wezterm $BACKUP_DIR/ 2>/dev/null
cp ~/.config/kitty $BACKUP_DIR/ 2>/dev/null

tar -czf $BACKUP_DIR.tar.gz $BACKUP_DIR
echo "Backup saved to $BACKUP_DIR.tar.gz"
```
